Mesh.Attrib Class

Module: kami-mesh-buffer

Mesh vertex attribute holder.

Location is optional and for advanced users that want vertex arrays to match across shaders. Any non-numerical value will be converted to null, and ignored. If a numerical value is given, it will override the position of this attribute when given to a mesh.

Parameters:

  • name String

    the name of the attribute

  • numComponents Number

    the number of components, e.g. 2 for vec2

  • location Number

    optional attribute index location

  • type Number

  • normalize Number

    whether to normalize to 0-1, default false
